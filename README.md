# Principles of Programming Final Project - Neural Network in Python

Thanks for looking at our simple neural network project. The main Network class is contained in Network.py.

The only dependency for the Network class to function is numpy and activation.py (found in the project root). If you would like to test the module out for yourself, just download network.py and activation.py and ensure that numpy exists in your environment.

For examples of usage of the network, see testing_main.ipynb. This is a jupyter notebook that we used throughout testing and contains clear examples of the Network's capabilities. To use testing_main.ipynb yourself, ensure that you have at least one of the datasets (found in datasets/) downloaded.
